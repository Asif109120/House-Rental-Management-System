# House Rental Management System (HRMS)

A comprehensive web-based solution for managing rental properties, designed for landlords, tenants, and administrators.

## Features
- User Management (Landlords, Tenants, Admins)
- Property Listings Management
- Search & Booking System
- Communication System
- Maintenance Request Tracking
- Reports & Analytics
- Responsive Design

## Technologies Used
- Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
- Backend: PHP
- Database: MySQL
- Additional: jQuery, Font Awesome

## Installation
1. Clone the repository to your xampp/htdocs directory
2. Import the database.sql file into MySQL
3. Configure database connection in config/db.php
4. Access the application through your web browser
